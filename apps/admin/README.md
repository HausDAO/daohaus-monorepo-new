# DAOhaus Admin

Core DAO interface for DAO administration:

- DAO overview data and settings management
- Proposal creation, voting and processing
- Member management
- Safe lists and token transfer proposals

## Development

1. Copy and fill in .env values:
   `cp .env.sample .env`

2. Run locally:
   `nx run admin:serve`
